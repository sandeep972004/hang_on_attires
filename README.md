# HangOn Attires — MERN Starter

This repository is a frontend React + Vite project for a clothes shop named "HangOn Attires".

Structure:

- `client/` — React + Vite frontend (main app)

Quick setup (client):

```powershell
cd client
npm install
npm run dev
```

Build for production:

```powershell
cd client
npm run build
```

Deployment notes:
- Frontend: ready for Vercel (build using `npm run build` in `client`). Set the output directory to `dist`.

See `client/README.md` for more details.
